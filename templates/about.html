FAQ

Why do we I have to use my own API Key? Isn't that dangerous?

Yes, I know it's slightly unconventional, and usually not good to share your API Key with others, however, I do not store or steal your API Key in any way (and if you are suspicious that I will you can use the local version found on my GitHub or run this web version on your own machine) and also I do not have a production key, so the least rate limiting way for everyone is to use your own key. I will attempt to get a production key in the future.

How do I get an API key?

Go to https://developer.riotgames.com/ and log in / register for an account, go to your dashboard and at the bottom there should be an option to generate an API key.
